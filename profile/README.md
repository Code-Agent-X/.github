## About

**Code Agent X** is an open-source project for building websites.

We are working on tools to design, build, and ship web pages more easily.

---

## Repositories

| Repository | Language | Description |
| ---------- | -------- | ----------- |
| [ChromaUI](https://github.com/Code-Agent-X/ChromaUI) | TypeScript | UI components for the builder |
| [Vector-Builder](https://github.com/Code-Agent-X/Vector-Builder) | HTML | Website builder |
| [Agent-Prototype](https://github.com/Code-Agent-X/Agent-Prototype) | Python | Early experiments |
